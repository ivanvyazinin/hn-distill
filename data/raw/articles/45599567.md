## What is the Fact Graph?

[](#what-is-the-fact-graph)

The Fact Graph is a production-ready knowledge graph for modeling, among other things, the United States Internal Revenue Code and related tax law. It can be used in JavaScript as well as any JVM language (Java, Kotlin, Scala, Clojure, etc.).

## Onboarding and Set Up

[](#onboarding-and-set-up)

See [ONBOARDING.md](/IRS-Public/fact-graph/blob/main/ONBOARDING.md) for environment/developer setup.

See [the Fact Graph 3.1 ADR](/IRS-Public/fact-graph/blob/main/docs/fact-graph-3.1-adr.md) for more information about the fact graph and how it has been changed since early 2025 See [here](/IRS-Public/fact-graph/blob/main/docs/from-3.0-to-3.1.md) for a brief description of changes between the older versions of the Fact Graph and the current v3.1 in this repository
