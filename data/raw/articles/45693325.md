# Bring Your 3D Models to Life

_Mesh2Motion_ is a FREE & open-source web application to animate your 3D models.  
Supports humanoid, four-legged, and bird creatures.

[Try it out](https://app.mesh2motion.org)

![Screenshot of the application](./images/animal-example.png)

### 🥳 Human & Animal rigs

*   Supports importing GLB, GLTF, and FBX models
*   Human and animal skeleton options
*   Intuitive skeleton positioning
*   Undo/Redo system when you make mistakes

### ✔️ Export Animations

*   Export multiple animations at once
*   Uses widely-supported GLB format
*   Human animation library from [Quaternius](https://quaternius.com/)

## FREE & Open-Source

Mesh2Motion is an open-source project. With the way 3d animations and modeling tools are progressing, there just needs to be some tool like this that is open-source that can evolve. The goal of this project is to provide a free and easy way to animate 3D models for web and game engines. Everything should be freely available for both personal and commercial projects. Check out the [GitHub repository](https://github.com/scottpetrovic/mesh2motion-app/) for all the code.
