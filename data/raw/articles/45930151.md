## About AI World Clocks

Every minute, a new clock is displayed that has been generated by nine different AI models.

Each model is allowed 2000 tokens to generate its clock. Here is its prompt:

`Create HTML/CSS of an analog clock showing ${time}. Include numbers (or numerals) if you wish, and have a CSS animated second hand. Make it responsive and use a white background. Return ONLY the HTML/CSS code with no markdown formatting.`

Created by [Brian Moore](https://brianmoore.com). You can also follow him on [Instagram](https://instagram.com/lanewinfield). Idea inspired by [Matthew Rayfield](https://matthew.rayfield.world/).
