# App Showcase

## [`atuin`](https://github.com/atuinsh/atuin)

[Section titled “atuin”](#atuin)

Atuin replaces your existing shell history with a SQLite database, and records additional context for your commands.

![atui demo](https://github.com/atuinsh/atuin/blob/main/demo.gif?raw=true)

* * *

## [`bandwhich`](https://github.com/imsnif/bandwhich)

[Section titled “bandwhich”](#bandwhich)

This is a CLI utility for displaying current network utilization by process, connection and remote IP/hostname

![bandwhich demo](https://github.com/imsnif/bandwhich/blob/main/res/demo.gif?raw=true)

* * *

## [`binsider`](https://github.com/orhun/binsider)

[Section titled “binsider”](#binsider)

Perform binary analysis in your terminal.

![binsider demo](https://github.com/orhun/binsider/blob/main/website/src/content/assets/quickstart.gif?raw=true)

* * *

## [`bottom`](https://github.com/ClementTsang/bottom)

[Section titled “bottom”](#bottom)

A customizable cross-platform graphical process/system monitor for the terminal

![bottom demo](https://github.com/ClementTsang/bottom/blob/main/assets/demo.gif?raw=true)

* * *

## [`crossword`](https://github.com/matrixfrog/crossword)

[Section titled “crossword”](#crossword)

Play crossword puzzles in your terminal.

![crossword demo](https://github.com/user-attachments/assets/8a2bb53b-f461-4ddd-b60c-9ae5170b501d)

* * *

## [`csvlens`](https://github.com/YS-L/csvlens)

[Section titled “csvlens”](#csvlens)

`csvlens` is A command line CSV file viewer. It is like less but made for CSV.

![csvlens demo](/csvlens.gif)

* * *

## [`dua`](https://github.com/Byron/dua-cli)

[Section titled “dua”](#dua)

`dua` is a disk space analysis tool designed for speed, leveraging parallel processing to quickly provide detailed disk usage information and allowing for faster deletion of unnecessary data compared to the standard ‘rm’ command.

[![dua demo](https://asciinema.org/a/kDnXUOeqBxZVMoWuFNqzfpeey.svg)](https://asciinema.org/a/kDnXUOeqBxZVMoWuFNqzfpeey)

* * *

## [`fzf-make`](https://github.com/kyu08/fzf-make)

[Section titled “fzf-make”](#fzf-make)

A command line tool that executes make target using fuzzy finder with preview window

![fzf-make demo](https://raw.githubusercontent.com/kyu08/fzf-make/main/static/demo.gif)

* * *

## [`gitui`](https://github.com/extrawurst/gitui)

[Section titled “gitui”](#gitui)

TUI for git written in rust

![gitui demo](https://github.com/extrawurst/gitui/blob/master/demo.gif?raw=true)

* * *

## [`gpg-tui`](https://github.com/orhun/gpg-tui)

[Section titled “gpg-tui”](#gpg-tui)

gpg-tui is a Terminal User Interface for GnuPG.

![gpg-tui demo](https://github.com/orhun/gpg-tui/blob/master/demo/gpg-tui-scrolling_rows.gif?raw=true)

* * *

## [`joshuto`](https://github.com/kamiyaa/joshuto)

[Section titled “joshuto”](#joshuto)

Ranger-like terminal file manager written in Rust

![joshuto demo](https://github.com/kamiyaa/joshuto/raw/main/screenshot.png?raw=true)

* * *

## [`material`](https://crates.io/crates/material)

[Section titled “material”](#material)

A material design color palette for the terminal.

![material demo](https://i.ibb.co/2MDKmh7/Screenshot-2022-08-02-at-16-43-12.png)

* * *

## [`minesweep-rs`](https://github.com/cpcloud/minesweep-rs)

[Section titled “minesweep-rs”](#minesweep-rs)

A mine sweeping game written in Rust

![minesweep-rs demo](https://github.com/cpcloud/minesweep-rs/blob/main/demo.gif?raw=true)

* * *

## [`oatmeal`](https://github.com/dustinblackman/oatmeal/)

[Section titled “oatmeal”](#oatmeal)

Oatmeal is a terminal UI chat application that speaks with LLMs, complete with slash commands and fancy chat bubbles. It features agnostic backends to allow switching between the powerhouse of ChatGPT, or keeping things private with Ollama. While Oatmeal works great as a stand alone terminal application, it works even better paired with an editor like Neovim!

![oatmeal-demo](https://github.com/dustinblackman/oatmeal/assets/5246169/9ee5e910-4eff-4deb-8065-aeab8bfe6b00)

* * *

## [`oha`](https://github.com/hatoo/oha/)

## [`television`](https://github.com/alexpasmantier/television)

[Section titled “television”](#television)

Television is a fast and versatile fuzzy finder TUI.

It lets you quickly search through any kind of data source (files, git repositories, environment variables, docker images, you name it) using a fuzzy matching algorithm and is designed to be easily extensible.

![television demo](https://github.com/user-attachments/assets/7a967f9c-779e-4915-baa8-160f586f8936)

## [`trippy`](https://github.com/fujiapple852/trippy)

[Section titled “trippy”](#trippy)

A network diagnostic tool that combines the functionality of traceroute and ping and is designed to assist with the analysis of networking issues.

![trippy demo](https://raw.githubusercontent.com/fujiapple852/trippy/master/assets/0.12.0/demo.gif?raw=true)

## [`yazi`](https://yazi-rs.github.io)

## [`yozefu`](https://github.com/MAIF/yozefu)

[Section titled “yozefu”](#yozefu)

Yōzefu is an interactive TUI application for exploring data of a Kafka cluster.

It is an alternative tool to AKHQ, Redpanda Console, or the Kafka plugin for JetBrains IDEs. It includes a search query language inspired by SQL, providing fine-grained filtering capabilities.

![yozefu demo](https://vhs.charm.sh/vhs-UpIJD2h92vKkj01XSS0r0.gif)
