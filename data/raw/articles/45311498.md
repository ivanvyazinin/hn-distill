# Introduction to Programming Languages

[Download the latest edition of the book](/pdfs/itpl-2023-08-10.pdf)

The book _Introduction to Programming Languages_ is an introductory book on programming languages. We, [Jaemin Hong](/) and [Sukyoung Ryu](https://plrg.kaist.ac.kr/ryu), write this book as the textbook of the KAIST _Programming Languages_ course. However, we do not limit the use of the book; we would be happy to see the book being used by anyone who wants to learn or teach basic programming language concepts, including syntax, semantics, type systems, and interpreter/type-checker implementations. If you are a faculty and want to use the book in your lecture, we politely ask you to introduce the authors and this webpage to your students. We would be delighted to receive comments and corrections, which may be sent to [jaemin.hong@kaist.ac.kr](mailto:jaemin.hong@kaist.ac.kr).
