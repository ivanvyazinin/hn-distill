# Simple trick to increase coverage: Lying to users about signal strength

[10/10/2025](https://nickvsnetworking.com/simple-trick-to-increase-coverage-lying-to-users-about-signal-strength/ "Permalink to Simple trick to increase coverage: Lying to users about signal strength")[Mobile Networks](https://nickvsnetworking.com/category/plmn/)[Android](https://nickvsnetworking.com/tag/android/), [Carrier Debug](https://nickvsnetworking.com/tag/carrier-debug/), [Carrier Privileges](https://nickvsnetworking.com/tag/carrier-privileges/), [RAN](https://nickvsnetworking.com/tag/ran/)[Nick](https://nickvsnetworking.com/author/nick/ "View all posts by Nick")

Poking around in Android the other day [I found this nugget in Carrier Config manager](https://android.googlesource.com/platform//frameworks/base/+/7a964f2199474ee99cc4a69e1654925ef4c4c09c/telephony/java/android/telephony/CarrierConfigManager.java#1156); a flag (KEY\_INFLATE\_SIGNAL\_STRENGTH\_BOOL) to always report the signal strength to the user as one bar higher than it really is.

![](https://nickvsnetworking.com/wp-content/uploads/2025/06/image-2-1024x233.png)

It’s [not documented in the Android docs](https://developer.android.com/reference/android/telephony/CarrierConfigManager), but it’s there in the source available for any operator to use.

Notably both [AT&T](https://github.com/AOSPA/android_packages_apps_CarrierConfig/blob/bf2410198baf884c05d3ccac36bdcff6aceaba9f/assets/carrier_config_carrierid_1187_AT%26T.xml#L74) and [Verizon](https://github.com/AOSPA/android_packages_apps_CarrierConfig/blob/bf2410198baf884c05d3ccac36bdcff6aceaba9f/assets/carrier_config_carrierid_1839_Verizon-Wireless.xml#L171) have this flag enabled on their networks, I’m not sure who was responsible for requesting this to be added to Android, nor could I find it in the `git-blame`, but we can see it in the [CarrierConfig](https://nickvsnetworking.com/android-carrier-privileges/) which [contains all the network settings](https://nickvsnetworking.com/android-apn-master-config/) for each of these operators.

![](https://nickvsnetworking.com/wp-content/uploads/2025/10/image-5-1024x532.png)

Operators are always claiming to have the biggest coverage or the best network, but stuff like this, along with the fake 5G flags, don’t help build trust, especially [considering the magic mobile phone antennas](https://nickvsnetworking.com/using-fancy-test-kit-to-measurecell-antenna-stickers/) which negate the need for all this deception anyway.
