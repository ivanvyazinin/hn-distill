# Start of BART service delayed Friday, September 5

A computer equipment problem following network upgrade work is preventing the start of service this morning. Seek alternative means of transportation. [bart.gov/alternatives](https://planner.bart.gov/?products=364#!P|TP!H|96581) provides options without BART service.
