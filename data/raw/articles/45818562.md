# Welcome to Hypothesis![¶](#welcome-to-hypothesis "Link to this heading")

Hypothesis is the property-based testing library for Python. With Hypothesis, you write tests which should pass for all inputs in whatever range you describe, and let Hypothesis randomly choose which of those inputs to check - including edge cases you might not have thought about. For example:

from hypothesis import [given](reference/api.html#hypothesis.given "hypothesis.given"), strategies as st

[@given](reference/api.html#hypothesis.given "hypothesis.given")([st.lists](reference/strategies.html#hypothesis.strategies.lists "hypothesis.strategies.lists")([st.integers](reference/strategies.html#hypothesis.strategies.integers "hypothesis.strategies.integers")() | [st.floats](reference/strategies.html#hypothesis.strategies.floats "hypothesis.strategies.floats")()))
def test\_sort\_correct(lst):
    \# hypothesis generates random lists of numbers to test
    assert my\_sort(lst) \== [sorted](https://docs.python.org/3/library/functions.html#sorted "sorted")(lst)

test\_sort\_correct()
