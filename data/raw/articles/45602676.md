### Description

Liquibase has migrated to the Functional Source License, which is not an open source license, as Liquibase itself [acknowledges](https://www.liquibase.com/blog/liquibase-community-for-the-future-fsl). But this repository continues to misleadingly characterize Liquibase as an open source project, particularly in various places in the file [README.md](https://github.com/liquibase/liquibase/blob/master/README.md).

### Steps To Reproduce

View [github.com/liquibase/liquibase](https://github.com/liquibase/liquibase), or the contents of the README.md file, to find out about the liquibase community project.

### Expected/Desired Behavior

The README.md file and any similar project documentation will no longer misleadingly suggest that liquibase is still an open source project.
