# Visopsys

### Welcome!

Visopsys is an alternative operating system for PC compatible computers. In development since 1997, this system is small, fast, and open source. It features a simple but functional graphical interface, pre-emptive multitasking, and virtual memory. Though it attempts to be compatible in a number of ways, Visopsys is not a clone of any other operating system. You can demo the distribution from a “live” USB stick, CD/DVD, or floppy disk … ([read more](https://visopsys.org/about/)).

* * *

#### Features of Visopsys[![dlbig](https://visopsys.org/wp-content/uploads/2016/09/dlbig.jpg)](https://visopsys.org/download/)

1.  Small & fast
2.  Graphical user interface
3.  Fully multitasking
4.  100% protected mode
5.  Open source, free software
