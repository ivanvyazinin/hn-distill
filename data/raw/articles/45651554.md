# normalize.css

## A cross-browser CSS foundation

[normalize.css](https://github.com/csstools/normalize.css/) makes browsers render all elements more consistently and in line with modern standards. It precisely targets only the styles that need normalizing.

[**Download** v11.0.0](https://csstools.github.io/normalize.css/11.0.0/normalize.css)

Chrome, Edge, Firefox ESR+, IE 9+, Safari 8+, Opera

[See the CHANGELOG](https://github.com/csstools/normalize.css/blob/11.0.0/CHANGELOG.md)

```
npm install @csstools/normalize.css
```
