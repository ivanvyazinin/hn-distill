# v0.26.0

## Features

*   Add build for windows/ARM64 platform. [#3190](https://github.com/sharkdp/bat/pull/3190) ([@alcroito](https://github.com/alcroito))
*   Add paging to `--list-themes`, see PR [#3239](https://github.com/sharkdp/bat/pull/3239) ([@einfachIrgendwer0815](https://github.com/einfachIrgendwer0815))
*   Support negative relative line ranges, e.g. `bat -r :-10` / `bat -r='-10:'`, see [#3068](https://github.com/sharkdp/bat/pull/3068) ([@ajesipow](https://github.com/ajesipow))
*   Support context in line ranges, e.g. `bat -r 30::5` / `bat -r 30:40:5`, see [#3345](https://github.com/sharkdp/bat/pull/3345) ([@cavanaug](https://github.com/cavanaug))
*   Add built-in 'minus' pager, e.g. `bat --pager=builtin` see PR [#3402](https://github.com/sharkdp/bat/pull/3402) ([@academician](https://github.com/academician))

## Bugfixes

*   Fix UTF-8 BOM not being stripped for syntax detection, see [#3314](https://github.com/sharkdp/bat/issues/3314) ([@krikera](https://github.com/krikera))
*   Fix `BAT_THEME_DARK` and `BAT_THEME_LIGHT` being ignored, see issue [#3171](https://github.com/sharkdp/bat/issues/3171) and PR [#3168](https://github.com/sharkdp/bat/pull/3168) ([@bash](https://github.com/bash))
*   Prevent `--list-themes` from outputting default theme info to stdout when it is piped, see [#3189](https://github.com/sharkdp/bat/pull/3189) ([@einfachIrgendwer0815](https://github.com/einfachIrgendwer0815))
*   Rename some submodules to fix Dependabot submodule updates, see issue [#3198](https://github.com/sharkdp/bat/issues/3198) and PR [#3201](https://github.com/sharkdp/bat/pull/3201) ([@victor-gp](https://github.com/victor-gp))
*   Make highlight tests fail when new syntaxes don't have fixtures PR [#3255](https://github.com/sharkdp/bat/pull/3255) ([@dan-hipschman](https://github.com/dan-hipschman))
*   Fix crash for multibyte characters in file path, see issue [#3230](https://github.com/sharkdp/bat/issues/3230) and PR [#3245](https://github.com/sharkdp/bat/pull/3245) ([@HSM95](https://github.com/HSM95))
*   Add missing mappings for various bash/zsh files, see PR [#3262](https://github.com/sharkdp/bat/pull/3262) ([@AdamGaskins](https://github.com/AdamGaskins))
*   Send all bat errors to stderr by default, see [#3336](https://github.com/sharkdp/bat/pull/3336) ([@JerryImMouse](https://github.com/JerryImMouse))
*   Make --map-syntax target case insensitive to match --language, see [#3206](https://github.com/sharkdp/bat/pull/3206) ([@keith-hall](https://github.com/keith-hall))
*   Correctly determine the end of the line in UTF16LE/BE input [#3369](https://github.com/sharkdp/bat/pull/3369) ([@keith-hall](https://github.com/keith-hall))
*   `--style=changes` no longer prints a two-space indent when the file is unmodified, see issue [#2710](https://github.com/sharkdp/bat/pull/2710) and PR [#3406](https://github.com/sharkdp/bat/pull/3406) ([@jyn514](https://github.com/jyn514))
*   Add missing shell completions, see [#3411](https://github.com/sharkdp/bat/pull/3411) ([@keith-hall](https://github.com/keith-hall))
*   Execute help/version/diagnostic commands even with invalid config/arguments present, see [#3414](https://github.com/sharkdp/bat/pull/3414) ([@keith-hall](https://github.com/keith-hall))
*   Fixed line numbers (`-n`) and style components not printing when piping output, see issue [#2935](https://github.com/sharkdp/bat/issues/2935) and PR [#3438](https://github.com/sharkdp/bat/pull/3438) ([@lmmx](https://github.com/lmmx))
