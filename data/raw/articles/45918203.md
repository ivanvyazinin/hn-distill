# [Microsoft confirms Windows 11 is about to change massively, gets enormous backlash](/news/microsoft-confirms-windows-11-is-about-to-change-massively-gets-enormous-backlash/ "Permalink to Microsoft confirms Windows 11 is about to change massively, gets enormous backlash")

[Sayan Sen](/profile/649440-sayan-sen/) Neowin [@ssc\_combater007](https://www.twitter.com/ssc_combater007) · Nov 13, 2025 00:44 EST · Hot! with [32 comments](#comments "View 32 Comments")

![Windows 11 wallpaper with thumbs down emoji next to it](https://cdn.neowin.com/news/images/uploaded/2025/06/1749990820_windows_11_thumbs_down_story.webp)

Recently, Neowin published an interview with AMD wherein the company suggested that its processors would be compatible with the next generation of Windows. The chip maker stated that its Ryzen AI PCs would be "not only compatible with future Windows capabilities but optimized for them." You can find the full interview [on this page](https://www.neowin.net/news/interview-amd-hints-at-ryzen-compatibility-with-next-major-windows-version-requirements/).

One of the reasons for mentioning Ryzen AI is because of how Windows itself is about to change in a really big way such that it already requires AI-specific hardware for certain features to work.

Microsoft's Windows chief Pavan Davuluri had earlier [hinted at such plans already](https://www.neowin.net/news/senior-microsoft-official-shares-what-next-major-windows-version-will-be-like/) about how the next evolution of OS will make it capable enough to make it "semantically understand you" as Windows will get "more ambient, more pervasive, more multi-modal". Using features like Copilot Vision it will be able to "look at your screen" and do more.

The company is working on [Model Context Protocol (MCP) for Windows 11](https://www.neowin.net/news/microsoft-is-bringing-model-context-protocol-to-windows-11-to-make-it-an-agentic-os/) in an effort to help it become the Agentic OS that essentially will turn Windows into an AI OS.

Here is how Microsoft puts it: "MCP on Windows offers a standardized framework for AI agents to connect with native Windows apps, enabling them to easily participate in agentic interactions on Windows. Windows apps can expose specific functionality to augment the skills and capabilities of agents installed locally on a Windows PC."

A few days ago, Davuluri shared his excitement about it on his official X handle. He seemed very eager to reveal what the company has in mind at the upcoming Ignite event regarding the agentic OS plans.

> Windows is evolving into an agentic OS, connecting devices, cloud, and AI to unlock intelligent productivity and secure work anywhere. Join us at [#MSIgnite](https://twitter.com/hashtag/MSIgnite?src=hash&ref_src=twsrc%5Etfw) to see how frontier firms are transforming with Windows and what’s next for the platform. We can’t wait to show you!…
> 
> — Pavan Davuluri (@pavandavuluri) [November 10, 2025](https://twitter.com/pavandavuluri/status/1987942909635854336?ref_src=twsrc%5Etfw)

Unfortunately for Microsoft and Davuluri, the response has been overwhelmingly negative, so much so that the comments on that X post have now been disabled. Interestingly, several of them have been heavily upvoted so it clearly shows many people do not like the sound of Windows becoming an agentic OS.

"How about making widows fast ? Not agentic" [said](https://x.com/G_melo_ding/status/1988075505573339282) one user clearly hoping Windows acted and felt faster than it now does. Another user [said](https://x.com/JanRingos/status/1988024015097667859) "You can't even correctly implement small taskbar icons, which is something users actually want. You are getting overwhelmingly negative feedback about all this AI stuff. And yet you persevere. Why?"

A user also suggested that all these extra features meant Windows would get more bloaty as they [wrote](https://x.com/headspawn/status/1988337980499804337) "Sounds like more bloat incoming? Have you considered making an OS that is performant and as bug free as possible? ... You guys seem to think everyone wants a hundred always on permanantly in beta features running and phoning home, eating up processing.."

Some of the commenters have also said that they are leaving Windows and Microsoft 365 products due to the overwhelming implementation of features that they may not want.

The comments on the thread have now been disabled as mentioned above, but you can tell your own thoughts about it in the article comments down below.
