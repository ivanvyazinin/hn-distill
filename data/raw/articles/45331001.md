# Maintain Your Codebase using AI

Making small changes like updating a single module version in many repositories can be daunting. Here we let you delegate these tasks to AI agents and let them create PRs for you.

## Features

![Prompt library](./assets/images/home/prompts2.png)

### Canvas

Using canvas, approve or edit affected items of your task. The list of repositories, file diffs, etc. can be modified by the canvas.

![Web search](./assets/images/home/search.png)

### InfraAsAI

Use our canvas and chatbot to share what's on your mind. Our agent will define a task and cook a bunch of PRs for you.

![Image generation](./assets/images/home/image.png)

### Pull requester

One task may require multiple PRs, we will help you create them and track them without the need to open IDE or any other dashboard.

![History](./assets/images/home/history.png)

### Rulebooks

Runbook for your agent. Define behavior based on rules and templates you prefer—how a commit message should look, or a Slack message asking for a review.

![Import content](./assets/images/home/import.png)

### Fine-tuned IaaC Models

Public language models are not designed to work with IaC-structured filesystems; they underperform with complex multi-repo organizations.

![Multilingual](./assets/images/home/multilingual.png)

### MCP config

Add internal and third party servers using a simple Yaml file.

## Built-in Agents

### KnowItAll

Discover your infrastructure from top to bottom. Indexing your org help us understand and connect the dots using our reasoning models.

### Patch Hero

We already indexed the patch notes of your dependencies to enrich generated codes.

### Github

We automate the process of git branching, making pull requests and taking care of issues.

### Jira

Mention your ticket directly in the chat and let the agent work on it. This agent can also create tickets with details out of suggested improvments provided by our agent.

### Slack

Let the exhausting work of asking for a review be done by our agent, we know which team is responsible for which repository.
