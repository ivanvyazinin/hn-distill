# Electron Apps Causing System-Wide Lag on Tahoe

[](#electron-apps-causing-system-wide-lag-on-tahoe)

See:

*   [electron/electron#48311 (comment)](https://github.com/electron/electron/issues/48311#issuecomment-3332181420)
*   [https://mjtsai.com/blog/2025/09/30/electron-apps-causing-system-wide-lag-on-tahoe/](https://mjtsai.com/blog/2025/09/30/electron-apps-causing-system-wide-lag-on-tahoe/)

Fixed versions:

*   36.9.2
*   37.6.0
*   38.2.0
*   39.0.0
*   and all above 39

This script detects apps with not yet updated versions of Electron.

## Temporary workaround:

[](#temporary-workaround)

Run

launchctl setenv CHROME\_HEADLESS 1

on every system start. The CHROME\_HEADLESS flag has a side effect of disabling Electron app window shadows, which makes them ugly, but also stops triggering the issue.

[**detect-electron-apps-with-macos-lag.sh**](#file-detect-electron-apps-with-macos-lag-sh)

mdfind "kind:app" 2>/dev/null | while read app; do

electronFiles=$(find "$app" -name "Electron Framework" -type f 2>/dev/null)

if \[\[ \-n "$electronFiles" \]\]; then

appName=$(basename "$app")

while IFS= read -r filename; do

electronVersion=$(strings "$filename" | grep "Chrome/" | grep -i Electron | grep -v '%s' | sort -u | cut -f 3 -d '/')

if \[\[ \-n "$electronVersion" \]\]; then

IFS='.' read -r major minor patch <<< "$electronVersion"

relativePath=$(echo "$filename" | sed "s|$app/||")

if \[\[ $major \-gt 39 \]\] || \\

\[\[ $major \-eq 39 && $minor \-ge 0 \]\] || \\

\[\[ $major \-eq 38 && $minor \-gt 2 \]\] || \\

\[\[ $major \-eq 38 && $minor \-eq 2 && $patch \-ge 0 \]\] || \\

\[\[ $major \-eq 37 && $minor \-gt 6 \]\] || \\

\[\[ $major \-eq 37 && $minor \-eq 6 && $patch \-ge 0 \]\] || \\

\[\[ $major \-eq 36 && $minor \-gt 9 \]\] || \\

\[\[ $major \-eq 36 && $minor \-eq 9 && $patch \-ge 2 \]\]; then

echo "✅ $appName: Electron $electronVersion ($relativePath)"

else

echo "❌ $appName: Electron $electronVersion ($relativePath)"

fi

break

fi

done <<< "$electronFiles"

fi

done
