# cppstat – C++ Compiler Support Status

[![github](resources/github-mark.svg)](https://github.com/cdervis/cppstat)

This site lists C++ features and their respective support by compilers and standard libraries, designed as a simple, quickly searchable table.

It's regularly updated and maintained as best as possible. If you see incorrect information or something missing, feel free to [contact me](mailto:cem@dervis.de?subject=C++ Compiler Support Change) or [submit a ticket](https://github.com/cdervis/cppstat/issues).

Explanation

Last updated on September 10, 2025 (changes)

*   Green: The feature is fully supported by the compiler / toolchain.
*   Orange: The feature is only **partially** supported by the compiler / toolchain.
*   Red: The feature is **not** supported by the compiler / toolchain (yet).
*   \[lib\]: The feature is a standard library feature (not a language feature).
*   You can click on the toolchain names to see more information about them, e.g. release dates and why a feature is partially supported.
