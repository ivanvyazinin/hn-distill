# µcad programming language

Welcome to the website of µcad!  
  
Microcad (or µcad) is a new open source programming language that can generate 2D sketches and 3D objects.  
  
The project is still in its early stages, but µcad is becoming increasingly stable. New ideas are being added to the code every week.  
  
In this blog, we want to keep you up to date on the latest developments at µcad.
