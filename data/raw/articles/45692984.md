## Getting Started

[](#getting-started)

To get a local copy up and running, please follow these simple steps.

1.  Clone the repo
    
    git clone https://github.com/linagora/twake-drive
    
2.  Run it with Docker
    
    cd tdrive
    docker compose -f docker-compose.minimal.yml up
    
3.  Open [http://localhost/](http://localhost/) in a browser

## Development

[](#development)

### Prerequisites

[](#prerequisites)

*   Node.js (Version: >=18.x)
*   MongoDB
*   Yarn _(recommended)_

### Setup

[](#setup)

1.  Launch MongoDB using
    
    docker run -p 27017:27017 -d mongo
    
2.  Launch frontend with
    
    cd tdrive/frontend/; yarn dev:start
    
3.  Launch backend with
    
    cd tdrive/backend/node/; SEARCH\_DRIVER=mongodb DB\_DRIVER=mongodb PUBSUB\_TYPE=local \\
    DB\_MONGO\_URI=mongodb://localhost:27017 STORAGE\_LOCAL\_PATH=/\[full-path-to-store-documents\]/documents \\
    NODE\_ENV=development yarn dev
    
    > If you need more parameters, create/edit `tdrive/backend/node/config/development.json` file
    
4.  The app will be running on port 3000
