# How-tos & Examples[¶](#how-tos-examples "Link to this heading")

## Delaying specific endpoints[¶](#delaying-specific-endpoints "Link to this heading")

Making an existing API slow can be easily accomplished combining mock’s [Base APIs](base_apis.html#base-api) and the [delay option.](cmd_reference.html#cmd-options-reference-delay)

$ mock serve \-p 8000 \--base example.com \--delay 2000

You may want however to make a specific endpoint slow instead of the whole API. This can be achieved using [middlewares](middlewares.html#middlewares):

$ mock serve \-p 8000 \--base example.com \--middleware '
if \[ "${MOCK\_REQUEST\_ENDPOINT}" = "some/endpoint" \]
then
    sleep 2 # wait two seconds
fi
'

With that last example, our API at `localhost:8000` will act as a proxy to `example.com`. All requests will be responded immediately except `some/endpoint` which will have a delay of 2 seconds.

## An API powered by multiple languages[¶](#an-api-powered-by-multiple-languages "Link to this heading")

$ mock serve \-p 3000 \\
    \--route js \\
    \--exec '
node <<EOF | mock write
console.log("Hello from Node.js!")
EOF
' \\
    \--route python \\
    \--exec '
python3 <<EOF | mock write
print("Hello from Python!")
EOF
' \\
    \--route php \\
    \--exec '
php <<EOF | mock write
<?php
echo "Hello from PHP!\\n";
?>
EOF
'

Let’s test it:

$ curl localhost:3000/js
\# Prints out: Hello from Node.js!
$ curl localhost:3000/python
\# Prints out: Hello from Python!
$ curl localhost:3000/php
\# Prints out: Hello from PHP!

## A stateful API[¶](#a-stateful-api "Link to this heading")

$ export TMP\=$(mktemp)
$ printf "0" \> "${TMP}"

$ mock serve \-p 3000 \\
    \--route '/hello' \\
    \--exec '
printf "%s + 1\\n" "$(cat ${TMP})" | bc | sponge "${TMP}"

printf "This server has received %s request(s) so far." "$(cat '"${TMP}"')" | mock write
'

Let’s test it:

$ curl localhost:3000/hello
\# Prints out: This server has received 1 request(s) so far.
$ curl localhost:3000/hello
\# Prints out: This server has received 2 request(s) so far.
$ curl localhost:3000/hello
\# Prints out: This server has received 3 request(s) so far.
