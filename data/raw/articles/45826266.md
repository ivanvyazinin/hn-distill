# Dillo web browser

[](#dillo-web-browser)

Dillo is a multi-platform graphical web browser, known for its speed and small footprint, that is developed with a focus on personal security and privacy. It is built with the [FLTK 1.3 GUI toolkit](http://fltk.org).

Screenshot of the [Dillo Website](https://dillo-browser.github.io/) rendered in Dillo:

[![](/dillo-browser/dillo/raw/master/doc/dillo.png)](/dillo-browser/dillo/blob/master/doc/dillo.png)

To install Dillo follow the [installation guide](/dillo-browser/dillo/blob/master/doc/install.md).

This repository contains mostly the original code of Dillo with some minor patches. Additional patches or pull requests are welcome.

See also other related forks: [dillo-plus](https://github.com/crossbowerbt/dillo-plus), [dilloNG](https://github.com/w00fpack/dilloNG), [D+ browser](https://sourceforge.net/projects/dplus-browser/) and [Mobilized Dillo](https://www.toomanyatoms.com/software/mobilized_dillo.html).

Warning

As of December 2023, the host `dillo.org` is [no longer under control](https://dillo-browser.github.io/dillo.org.html) of Dillo developers. A copy of the old website is archived in [GitHub Pages](https://dillo-browser.github.io/old/) and the [Wayback Machine (May 2022)](http://web.archive.org/web/20220508022123/https://www.dillo.org/).
