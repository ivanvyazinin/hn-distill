# Agent Autocomplete, Hooks, and Team Rules

### [Autocomplete for Agent](#autocomplete-for-agent)

When writing prompts, autocomplete suggestions will appear based on recent changes. Tab to accept suggestions and attach files to context.

### [Hooks (beta)](#hooks-beta)

You can now observe, control, and extend the Agent loop using custom scripts. Hooks give you a way to customize and influence Agent behavior at runtime.

Use Hooks to audit Agent usage, block commands, or redact secrets from context. It's still in beta and we'd love to hear your feedback.

### [Team rules](#team-rules)

Teams can now define and share global rules from the dashboard that will be applied to all projects. We’ve also shipped team rules for [Bugbot](https://cursor.com/bugbot), so behavior is consistent across repos.

![](https://cdn.sanity.io/images/2hv88549/production/12fedd8c2375c64f392d537a0b868402369f41ea-2892x2286.png?auto=format)

### [Share prompts with deeplinks (beta)](#share-prompts-with-deeplinks-beta)

Generate shareable deeplinks for reusable prompts. Useful for setup instructions in documentation, team resources, and sharing workflows. See our [documentation](https://cursor.com/docs/integrations/deeplinks) for how to create them.

### [Monitor Agents from menubar](#monitor-agents-from-menubar)

Quickly check the status of Cursor Agents right from your menubar.

### [Image file support for Agent](#image-file-support-for-agent)

Agent can now read image files directly from your workspace and include them in context. Previously, only pasted images were supported.
