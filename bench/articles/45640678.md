# v0.26.0

## Features

*   Add build for windows/ARM64 platform. [#3190](https://github.com/sharkdp/bat/pull/3190) ([@alcroito](https://github.com/alcroito))
*   Add paging to `--list-themes`, see PR [#3239](https://github.com/sharkdp/bat/pull/3239) ([@einfachIrgendwer0815](https://github.com/einfachIrgendwer0815))
*   Support negative relative line ranges, e.g. `bat -r :-10` / `bat -r='-10:'`, see [#3068](https://github.com/sharkdp/bat/pull/3068) ([@ajesipow](https://github.com/ajesipow))
*   Support context in line ranges, e.g. `bat -r 30::5` / `bat -r 30:40:5`, see [#3345](https://github.com/sharkdp/bat/pull/3345) ([@cavanaug](https://github.com/cavanaug))
*   Add built-in 'minus' pager, e.g. `bat --pager=builtin` see PR [#3402](https://github.com/sharkdp/bat/pull/3402) ([@academician](https://github.com/academician))
